 GPA & CGPA Calculator System

This project calculates GPA for a semester and CGPA across semesters.

 Features
- Add courses with units and grades
- Automatic GPA calculation
- CGPA calculation

  
  Technologies Used
- Programming Language: C++
- Paradigm: Procedural Programming
- Platform: Console / Terminal
- Compiler: Code::Blocks


 Sample Usage
- Student enters number of semesters
- Enters courses, units, and grades
- System calculates GPA for each semester
- System calculates final CGPA
